Estrategia Digital


Herramientas Esenciales para un Desarrollo Exitoso


La estrategia digital es un plan que utiliza herramientas y tecnologías específicas para lograr objetivos comerciales. Para un desarrollo exitoso, se requieren herramientas como Google Analytics, Git, Web Assembly, Docker y Postman (pruebas de API), a fin de optimizar procesos, mejorar la experiencia del cliente y adaptarse a un mercado dinámico.

Google Analytics


Es una herramienta de análisis web que rastrea y reporta el tráfico y comportamiento de usuarios en sitios y aplicaciones. Se combina con Google Console y herramientas necesarias para los proyectos de promoción

Github


Es el recurso donde se guarda en repositorios las versiones de lo avanzado en un proyecto. Junto con Github Pages se puede disponer de una plataforma probadamente fidedigna. Github Actions permite automatizar testeos con Docker y Selenium

WebAssembly


WebAssembly (Wasm) es un formato de código binario que permite ejecutar aplicaciones a alta velocidad en navegadores web.

Postman - Talend


Para probar APIs, tenemos Postman, aunque también está la opción de Talend. Facilitando el envío de solicitudes HTTP y la visualización de respuestas.

Docker


Docker es una plataforma de contenedorización que empaqueta aplicaciones y sus dependencias en contenedores portables y aislados.

Web Workers - Service Workers


Entender elementos en el caché de navegador, local storage, coockies, mejora experiencia de usuario. Se puede hacer con un framework como React o Angular pero también de manera manual y tener un control más granular

Tecnologías a Disposición

Tecnologías Fundamentales

  • MySQL: Ampliamente utilizado y fácil de aprender para bases de datos.
  • PostgreSQL: Similar a MySQL, ofrece características avanzadas y es muy utilizado en entornos de aprendizaje.
  • Git: Herramienta esencial para el control de versiones, con una amplia documentación y uso generalizado.
  • Firebase: Plataforma fácil de usar para desarrollo de aplicaciones, especialmente en la web y móvil.
  • Cloudflare: Herramienta de CDN y seguridad fácil de entender y utilizar para principiantes.
  • API de Postman: Útil para probar APIs y muy accesible para quienes inician en el desarrollo de servicios web.
  • GitHub Pages: Ideal para principiantes que quieren publicar páginas web simples.
  • YAML: Lenguaje de marcado fácil de entender, utilizado en configuraciones.


Frontend

  • Aplicaciones web progresivas (PWA): Enfocadas en mejorar la experiencia del usuario en aplicaciones web.
  • AMP (Accelerated Mobile Pages): Optimización de páginas para cargas rápidas en móviles, relevante para el front-end.
  • Web Assembly: Herramienta que permite ejecutar código en el navegador, importante para desarrollo frontend avanzado.
  • Servicios web de RESTful: Comúnmente utilizados en el frontend para comunicarse con el backend.
  • GitHub Actions: Si bien es general, su uso puede orientarse hacia flujos de trabajo de desarrollo frontend.


Bien Técnico

  • Kubernetes: Complejo y técnico, utilizado para la orquestación de contenedores.
  • TensorFlow: Herramienta técnica para machine learning y deep learning.
  • Grafana y Prometheus: Herramientas de monitoreo y visualización de datos, muy técnicas.
  • Ansible y Terraform: Herramientas de automatización y gestión de infraestructura que requieren conocimientos técnicos.
  • Multithreading y Hilo de ejecución: Conceptos técnicos de programación avanzados.
  • Kali Linux: Orientado a pruebas de seguridad, técnico y específico.
  • Nmap: Herramienta técnica para escaneo de redes y seguridad.


Complementarios

  • Selenium: Útil para pruebas automatizadas, complementa el desarrollo de aplicaciones.
  • Talend: Herramienta de integración de datos que puede ser útil en varios contextos.
  • C# y PHP: Lenguajes de programación que pueden ser utilizados en diferentes entornos.
  • Web scraping y Rastreo web: Técnicas que pueden ser aplicadas en diversos contextos, especialmente en análisis de datos.
  • Fiddler y Wireshark: Herramientas complementarias para monitoreo y depuración de redes.
  • Windows Server y VMware: Herramientas de administración de servidores que son útiles en infraestructura.
  • Computación en la nube: Concepto amplio que puede aplicarse en múltiples áreas de IT.
